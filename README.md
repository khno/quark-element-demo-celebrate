## Installation


Use npm:
```
npm install --save quarkc-demo-celebrate
```

Use CDN:
```
<script src="https://fastly.jsdelivr.net/npm/quarkc@latest"></script>
<script type="module" src="https://unpkg.com/quarkc-demo-celebrate@latest"></script>
```

<!DOCTYPE html>
